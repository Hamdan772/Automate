# Changelog

All notable changes to Automate will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Automate (rebranded from Nanobrowser)
- Multi-agent AI system (Navigator, Planner, Validator)
- Support for multiple LLM providers (OpenAI, Anthropic, Gemini, Ollama)
- Natural language command interface
- Chrome Extension Manifest V3 support
- Side panel UI with React and Tailwind CSS
- Settings page for API configuration
- Multi-language support (English, Portuguese, Chinese)
- Privacy-focused local processing
- Open-source MIT license

### Changed
- Rebranded from "Nanobrowser" to "Automate"
- Updated all UI strings and documentation
- Modernized extension manifest

### Fixed
- Build system compatibility issues
- TypeScript type checking errors
- Extension export conflicts

## [0.1.12] - 2025-11-03

### Changed
- Complete rebrand to "Automate"
- Updated extension name and descriptions
- Fixed build process for production deployment
